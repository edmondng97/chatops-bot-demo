01 — PRINCIPLES · 核心理念

01

确定性优先

能用死流程做掉的绝不交给模型：解析、路由、向导、生命周期，全部跑在无 LLM 的门卫里，永远不会退化。

02

一线程一实例

一个 thread 就是一个命令实例，从生到死。状态严格按线程隔离，绝不串台，这是铁律。

03

config 驱动扩展

新增一个能力 = 一份 JSON + 一个 worker skill。路由和生命周期内核一行不动。
